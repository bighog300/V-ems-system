
# OpenEMR Customization Specification

## Scope
Defines all OpenEMR configurations and extensions.

## Custom Elements
- Forms: ambulance_assessment, handover_summary
- Roles: crew, supervisor
- API clients

## Deployment
- Use configuration scripts
- Prefer extensions over core changes

## Scripts
- setup-api-client.sh
- install-forms.sh
- configure-roles.sh
