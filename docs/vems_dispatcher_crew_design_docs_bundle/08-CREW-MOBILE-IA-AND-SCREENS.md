# Crew Mobile IA and Screen Model

## Proposed mobile flow

### Screen 1 — Current Assignment Home
Shows:
- current assignment status
- destination / address
- urgency
- crew/unit
- big primary next-step button

### Screen 2 — Incident Details
Compact essentials:
- incident type
- location
- dispatch notes
- hazards / warnings
- patient summary if available

### Screen 3 — Patient / Encounter
Minimal structured form:
- key identifiers
- condition summary
- interventions snapshot
- quick-add / structured action controls

### Screen 4 — Handover
Guided handover checklist:
- receiving site
- disposition
- required fields only
- final submit
