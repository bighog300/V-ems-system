# Crew Mobile Workflow UI Overview

## Purpose
Design a mobile-first workflow UI for ambulance crews using phones or tablets in the field.

## Primary users
- Paramedic
- EMT / ambulance crew member
- Crew lead

## Field realities to design for
- one-handed use
- touch interaction
- intermittent connectivity
- glare / outdoor use
- rapid transitions between states
- low time and attention budget

## Core crew goals
1. See current assignment immediately
2. Acknowledge and move through status steps quickly
3. Access the essential incident/patient details without scrolling through noise
4. Record key workflow milestones
5. Complete handover with minimal friction
