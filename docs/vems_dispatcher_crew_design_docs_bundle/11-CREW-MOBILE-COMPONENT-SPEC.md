# Crew Mobile Component Specification

## Components

### AssignmentHero
Top card showing:
- incident ID
- incident type
- address
- urgency
- status
- next action

### PrimaryActionBar
Persistent bottom action area with one dominant CTA.

### HandoverForm
Minimal required fields with large controls and clear validation.
